# Expense Tracker API

A RESTful Expense Tracker API built using Java and Spring Boot.

## Features

* Add a new expense
* Get all expenses
* Get expense by ID
* Update an existing expense
* Delete an expense

## Technologies Used

* Java
* Spring Boot
* REST APIs
* Maven
* Postman
* Git & GitHub

## API Endpoints

### Get All Expenses

GET /expenses

### Get Expense By ID

GET /expenses/{id}

### Add Expense

POST /expenses

### Update Expense

PUT /expenses/{id}

### Delete Expense

DELETE /expenses/{id}
